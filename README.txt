+──────────────────────────────────────────────────────────────────────────────────────────────────────────────+
| Build Instructions                                                                                           |
+──────────────────────────────────────────────────────────────────────────────────────────────────────────────+
| Install cppzmq by running:                                                                                   |
| $ sudo apt install libzmq3-dev                                                                               |
|                                                                                                              |
| Make sure CMake is installed and run:                                                                        |
| $ ./build.sh                                                                                                 |
|                                                                                                              |
| If everything executed without any errors, the client and server executables should now be in folder "build".|
+──────────────────────────────────────────────────────────────────────────────────────────────────────────────+
